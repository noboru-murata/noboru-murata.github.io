about this site

This site is produced and maintained by the following tools.

Hugo

[Hugo] (https://gohugo.io) is one of the most popular open-source static site generators. With its amazing speed and flexibility, Hugo makes building websites fun again.

For installing on macOS, it is easy to use Homebrew.

brew install hugo

Hugo theme hello friend NG

[Hello Friend NG] (https://themes.gohugo.io/hugo-theme-hello-friend-ng/) is a simple theme for Hugo, which was inspired by the [hello-friend] (https://themes.gohugo.io/hugo-theme-hello-friend/) and [hermit] (https://themes.gohugo.io/hermit/).

In this site, default setups are used without any modification.

ox-hugo

[ox-hugo] (https://ox-hugo.scripter.co) is an Org exporter backend that exports Org to Hugo-compatible Markdown (Blackfriday) and also generates the front-matter (in TOML or YAML format).

If use-package is installed, just add the below to the config.

(use-package ox-hugo :after ox)

Then, all the MD files are maintained in one org file for Emacs and exported with C-c C-e H A.
